# homebrew-tap

Personal Homebrew tap for Reader and other tools.

## Installation

```bash
brew tap joqk12345/homebrew-tap
brew install --cask reader
```

## Available Casks

- `reader`

## Upgrade

```bash
brew update
brew upgrade --cask joqk12345/tap/reader
```

If Homebrew says the latest version is already installed but the app appears outdated, force reinstall:

```bash
brew reinstall --cask joqk12345/tap/reader
brew info --cask joqk12345/tap/reader
```

## Uninstall

```bash
brew uninstall --cask reader
```
